# Luma3DS 日本語版

日本語版Luma3DS（3gxローダー付き）

### メニュー表示
<img src="https://i.imgur.com/noKbIBZ.jpg" width="480px">

### インストール方法

`boot.firm`をSDカードのルートに、`unifont_full.bin`をSDカードのlumaフォルダに配置します。

### 注意事項

1. `unifont_full.bin`はフォントファイルです。これが無いと正常な表示ができないため、必ず配置してください。
2. 商用利用は禁止されています。
